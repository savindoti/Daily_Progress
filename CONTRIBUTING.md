# Contributing

Thanks for your interest in improving the Daily Dashboard.

- Fork the repo and create a topic branch for your change.
- Keep changes small and focused; include tests when adding behavior.
- Run linters and the build locally before submitting a PR.
- Open a pull request describing the change and linking any related issues.

For major changes (history rewrite, large removals), please discuss in an issue first.

Maintainers: run `npm --prefix frontend install` and `pip install -r backend/requirements.txt` to test locally.
